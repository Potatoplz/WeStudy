<h2> 💎 주요 기능 </h2>

<b>1. 네이버, 카카오 로그인</b>
- API 적용, 기본 정보 불러오기

<b>2. 게시판 - REST API </b>
 - 모든 게시판 REST API 적용
 
<b>3. 카카오페이 결제</b>
 - 카카오페이 API 적용
 - 테스트 결제 가능

<b>4. 지도 API </b>
 - 등록된 카페 마커 기능
 - 클릭시 오버레이 
 
<b>5. 채팅 </b>
 - Firebase, stomp, 웹소켓 이용한 채팅 기능
 - 지난 대화 불러오기, 실시간 업데이트 가능

<b>6. 일정 관리 (달력) </b>
 - 일정 추가, 수정, 삭제 가능
 - 스터디룸 예약시 일정에 자동 추가
